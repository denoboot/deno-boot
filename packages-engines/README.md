# DenoBoot Engines

DenoBoot Engines are Plug and Play kernels for DenoBoot.

## Engines

* ### denoboot-oak-engine

This engine provides integration with the Oak web framework.

> More engines to come...


> For now, engines will live in the same repository as the core framework. In the future, they will be moved to their own repositories. Official engines on DenoBoot github org and community engines on denoboot-community github org. Developers can also create their own engines and publish them.
